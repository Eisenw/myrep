# 3. 针对观察记录进行评分

## api

### post方法

/api/observation/score

### 参数

```
studentId: 学生id
level: 水平(打分)
classId:班级id
observationId:观察记录id
itemCode:评分项代码
```

## 示例

/api/observation/score?studentId=37e7df0f-9b43-4533-9d08-1dbe5c988bac&level=0&classId=f057359a-0ffb-4509-b194-129fb732994e&observationId=328&itemCode=11

```
studentId:37e7df0f-9b43-4533-9d08-1dbe5c988bac
level:0
classId:f057359a-0ffb-4509-b194-129fb732994e
observationId:328
itemCode:11
```



### 返回

```json
{
    "code": 200,
    "msg": "正常返回",
    "detail": "观察记录成功"
}
```
